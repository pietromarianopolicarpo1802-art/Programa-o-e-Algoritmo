## 📚 Programação e Algoritmo

---

## 📖 Introdução

A disciplina de **Programação e Algoritmo** tem como objetivo ensinar os fundamentos da programação e desenvolver o raciocínio lógico necessário para criar soluções computacionais. Por meio de exercícios e projetos, os estudantes aprendem a transformar problemas em algoritmos e programas funcionais.

---

## 🎯 O que é Programação e Algoritmo?

Programação é o processo de criar instruções para que um computador execute determinadas tarefas. Já um algoritmo é uma sequência lógica e organizada de passos utilizada para resolver um problema ou atingir um objetivo específico.

---

## 📚 Principais Conteúdos

### 🧠 Lógica de Programação

Desenvolvimento do pensamento lógico para analisar problemas e criar soluções eficientes.

### 📋 Algoritmos

Criação de sequências de instruções capazes de resolver desafios computacionais.

### 🔢 Variáveis e Tipos de Dados

Armazenamento e manipulação de informações dentro dos programas.

### ➕ Operadores

Utilização de operadores aritméticos, relacionais e lógicos para realizar cálculos e comparações.

### 🔀 Estruturas Condicionais

Tomada de decisões utilizando comandos como `if`, `else` e `switch`.

### 🔄 Estruturas de Repetição

Execução repetida de tarefas utilizando `for`, `while` e `do while`.

### ⚙️ Funções

Organização do código em blocos reutilizáveis responsáveis por tarefas específicas.

### 📊 Vetores e Matrizes

Estruturas de dados utilizadas para armazenar e organizar conjuntos de informações.

### 💻 Linguagem C++

Linguagem de programação utilizada para o desenvolvimento das atividades e projetos da disciplina.

Além disso, ela desenvolve habilidades essenciais como lógica, criatividade e resolução de problemas.
