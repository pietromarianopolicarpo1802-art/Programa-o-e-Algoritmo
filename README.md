# 📚 Programação e Algoritmo
Introdução

A disciplina de Programação e Algoritmo tem como objetivo ensinar os fundamentos da programação e desenvolver o raciocínio lógico necessário para criar soluções computacionais. Por meio dela, os estudantes aprendem a analisar problemas, planejar soluções e transformá-las em programas funcionais.

O que é Programação e Algoritmo?

Programação é o processo de criar instruções para que um computador execute determinadas tarefas. Já um algoritmo é uma sequência organizada de passos utilizada para resolver um problema ou alcançar um objetivo específico.

Principais Conteúdos
🧠 Lógica de Programação

Desenvolvimento do pensamento lógico para resolver problemas de forma eficiente.

📋 Algoritmos

Criação de sequências de instruções para solucionar desafios computacionais.

🔢 Variáveis e Tipos de Dados

Armazenamento e manipulação de informações dentro dos programas.

➕ Operadores

Utilização de operadores aritméticos, relacionais e lógicos para realizar cálculos e comparações.

🔀 Estruturas Condicionais

Tomada de decisões utilizando comandos como if, else e switch.

🔄 Estruturas de Repetição

Execução repetida de tarefas com for, while e do while.

⚙️ Funções

Organização do código em blocos reutilizáveis que executam tarefas específicas.

📊 Vetores e Matrizes

Estruturas de dados utilizadas para armazenar e organizar grandes quantidades de informações.

💻 Linguagem C++

Linguagem utilizada para o desenvolvimento das atividades e projetos da disciplina.

Conclusão
